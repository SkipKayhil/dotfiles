# dotfiles
## Install
```
git clone http://github.com/skipkayhil/dotfiles ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```
The bootstrap script symlinks all the `.symlink` files and then prompts to install [dein.vim](http://github.com/Shougo/dein.vim)
